## What Does EDF: MML Offer?:

- #### Configuration and Customization:
EDF: MML uses a set of tools and data files to configure and add custom mission packs (coming soon), weapon data, and UI text setups. Whether you want to enhance your offline or online experience, this tool makes it possible to tweak existing parameters, add new guns, class categories, and patch supported content.
- #### Effortless Mod Combining:
With EDF: MML, you can merge multiple mods as long as they include a Mod_config_data.json file. This configuration allows for precise integration of new content without manual merging hassles.
- #### Manifest-Based Uninstallation:
The DirManifestToFilesUninstaller feature manages mod removal, ensuring clean and safe uninstallation by protecting essential game directories.
- #### Build and Repair Tables:
Easily build or repair data tables with just a click. Keep your game stable and organized even when multiple mods are in play.
- #### Toggle Modloader and Manage Updates:
Effortlessly toggle the mod loader on or off, check for the latest updates, and ensure your tool and mods are always up to date.
- #### Launch Games and Access Saves:
Quickly launch EDF games directly from the tool or access save folders for EDF 4.1, 5, and 6 with ease

- #### Setup Requirements:
Each mod needs a *YourModName*Mod_config_data.json file placed in the appropriate directory. 

    EARTH DEFENSE FORCE 6\Mods\EDF 6 MOD SETTINGS MAKER\MOD CONFIG DATA PLACED HERE\*YourModName*Mod_config_data.json
This file allows the mod loader to recognize and properly configure each mod's settings.
##### However, this location might change based on community feedback or updates to the modding tool. The EDF community's input is valuable in deciding the best practices for organizing and managing mod configurations,
- #### Installation and Uninstallation:
Use the manifest to guide mod management. Before finalizing any changes, a manual review helps ensure safe and correct modifications.
- #### Error Management and Logs:
The error block feature displays logs related to the modding process, allowing you to troubleshoot effectively. You can clear these logs anytime to keep your workspace clean and organized.

## Future Features and Expansions:
- #### Support for Custom Mission Packs and More:
Upcoming updates will include support for adding custom mission packs, further expanding the game's replayability.
- #### Expanding Compatibility:
While currently optimized for EDF 6, future updates aim to support EDF 4.1 and EDF 5, making EDF: MML a comprehensive modding solution for the entire series on PC so far.
